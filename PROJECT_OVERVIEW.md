# US ❤️

## Project Overview

US ❤️ is a premium long-distance relationship application designed to help couples stay emotionally connected through shared memories, interactive experiences, planning tools, entertainment, and a persistent virtual world.

The application uses:

- Node.js
- Express.js
- Supabase
- Three.js
- HTML
- CSS
- JavaScript
- Progressive Web App (PWA)

The project follows a modular architecture where every major feature is implemented independently while sharing authentication, storage, and realtime services.

# Design Principles

- Modular architecture
- Reusable components
- Mobile-first responsive design
- Premium animations
- Three.js for immersive 3D experiences

- Offline-capable PWA
- Production-ready code

---

# Development Rule

Every new feature must:

- Reuse the existing architecture.
- Avoid rewriting completed modules.
- Extend the application through modular additions.
- Preserve backward compatibility.
- Maintain responsive performance on desktop and mobile.
